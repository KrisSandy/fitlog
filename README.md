# FitLog 💪

Personal workout tracker + AI coach. Built with Next.js + TypeScript.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Pages

- `/` — Home
- `/routine` — Weekly routine chart
- `/log` — Log a workout (chat UI)

## Next Steps

- Add MCP server in `/mcp-server` folder
- Wire up Claude API for real-time AI feedback after each logged set
